# Филимонский Ровер
Код для игры "Филимонский Ровер"

В игре вы играете за программиста маленького ровера. Каждый уровень представляет из себя дорожку с стартом и финишем. Ваша цель — написать программу на языке Python, с помощью которой ровер сможет дойти до конца дорожки, не съезжая с неё в процессе. Игра создана для учеников 8-11 классов, которые хотят начать учится программированию.
